# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Reporting

If you experience or witness unacceptable behavior, please report it by opening a
[GitHub Issue](https://github.com/N283T/zsasa/issues) or contacting the maintainer directly.

All reports will be reviewed and investigated promptly and fairly.
